# Student Financials

The Student Financials section shows all student transactions and account balances, which is essential for visa documentation and financial verification.

## Overview

![Student Financials Overview](images\financials.png)
*Student financial transactions and balances*

This section provides a comprehensive view of all financial activity for the student.

## Key Use for ISS Office

:::tip J-2 Funding Verification
Primarily for the ISS office, we are mostly concerned with cross-referencing and verifying the **$10,000 amount transaction** for J-2 funding verification. This allows us to provide students with the respective documentation they need.
:::

## Available Financial Categories

The header contains many useful categories to help you sort through financial information:

### Main Categories

- **Academic Period** - View finances by semester/term
- **Student Financials Period Records** - Period-specific financial records
- **H-SF Additional Data** - Supplementary financial data
- **Outstanding Charges and Payments** - Current balances and pending payments
- **Transaction Summary** - Aggregated view of all transactions
- **Account Transactions** - Detailed individual transactions
- **Historical Transactions** - Past financial activity

### Finding the J-2 Funding Transaction

To locate the $10,000 J-2 funding amount:

1. Navigate to **Account Transactions** or **Transaction Summary**
2. Filter by the appropriate academic period
3. Look for the $10,000 transaction
4. Use this information to complete J-2 documentation

:::warning Verify Before Certifying
Always verify the funding amount and transaction date before providing official documentation to students for their visa applications.
:::
